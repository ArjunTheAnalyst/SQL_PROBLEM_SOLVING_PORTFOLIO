# 🧠 SQL Problem Solving Portfolio

## 📌 Overview

This repository contains a collection of SQL solutions covering business analytics, data cleaning, reporting, ranking, transaction monitoring, customer analysis, and advanced SQL problem-solving scenarios.

The projects demonstrate practical applications of SQL using Common Table Expressions (CTEs), Window Functions, Aggregations, Ranking Functions, Conditional Logic, Data Cleaning techniques, and Business KPI calculations.

Each solution is stored in its own SQL file and focuses on solving a specific business or analytical problem.
